SWF over EE6 web app for Jboss AS 7.1

This is a Spring Web Flow application that use springwebflow-optforrepl to take advantage of Jboss AS facility for session replication granularity.

Usage

Every time you click the link below, it will open a simple page in other tab creating a new flow execution per tab.
In a cluster environment only the session information related to the specific execution will be replicated.

  • New simple flow /flow/simple
